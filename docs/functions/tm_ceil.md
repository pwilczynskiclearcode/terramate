# `tm_ceil` Function

`tm_ceil` returns the closest whole number that is greater than or equal to the
given value, which may be a fraction.

## Examples

```
tm_ceil(5)
5
tm_ceil(5.1)
6
```

## Related Functions

* [`tm_floor`](./tm_floor.md), which rounds to the nearest whole number _less than_
  or equal.
